## INTRODUCTION

Security is a non-negotiable in big data systems. A customer needs to know their data is secure or they won't be able to use the application. Data governance is a framework of policies, roles, and standards that ensure an organization's data is accurate, secure, available, and usable throughout its lifecycle. In this module, you will learn best practices of data security and governance. Examples from real-world applications will be provided to understand how security and governance are implemented in practice.



## Learning Objectives

At the conclusion of this module, you should be able to:

* Explain best practices in data security and governance.
* Understand the major data security risks and how to mitigate them.
* Progress toward executing an end-to-end predictive modeling project using a large dataset.
